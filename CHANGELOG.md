# Changelog


## v1.1.1 (2025.11.30)
### Changed
- `Markdown.js`의 볼드/이탤릭/볼드+이탤릭 변환 시, 특수 괄호 부착 예외 조건에 이모지를 추가
- 각종 최적화: 실제 LLM의 응답을 1,000회 반복 변환하여 테스트했을 때 속도 향상이 있었음
  - `Markdown.js`: 약 66% 처리 속도 향상
  - `LaTeX.js`: LaTeX가 포함된 문자열의 속도 25%, 포함되지 않은 문자열의 처리 속도 80% 향상
  - `tableToImage`: 메모리 정리 강화, 미세한 속도 향상


## v1.1.0 (2025.11.24)
### Added
- `tableToImage.js` 추가 (Graal JS 전용)

### Changed
- `LaTeX.js`에서 정규식 개선
- `markdown.js`에서 구분선 변환 개선

### Fixed
- LaTeX 변환 시 `$$ ... $$` 블록 내에 `$ ... $` 패턴이 있으면 변환이 적용되지 않는 문제 수정
